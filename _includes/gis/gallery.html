Topics

Gisquick offers ready to use Topics (specialized layer compilations), which are suitable for basic users without any experiences with GIS. Topics are accessed from the Main panel.

Layers

Gisquick also offers well arranged layers (base and overlay layers linking up with attribute data tables and layer metadata) for users who are used to working with common GIS.

Tables

Attribute table tool allows to display data attributes of any layer with enabled WFS service. You can also display and zoom-to geometry of selected object in the table.

Filters

Attribute table allows to filter displayed objects by visible screen area, or by logical conditions, which can be specified for any column in the table.

Print preview

Print preview shows you a page layout with appearance very close to the final print output. Map will be printed at the same position, angle and zoom level that you set.

Print panel

Print tool with opened panel. It allows you to configure print output (format, resolution), select page layout, or enter text labels (if defined in selected layout).

Identification

When you want view attribute values for a feature, use the identify tool that can be accessed from the Tools menu.

Measurement

Measure tool allows you to measure point coordinates, distances and areas. Measure panel has also options to select measure units, zoom to measured object, and it may contain some secondary measurements (not displayed on the map).

Mobile

Application running on mobile device with Android OS. It offers the same features as web application on regular desktop, but use more compact layout and has additional account settings and projects manager.
