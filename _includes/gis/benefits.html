QGIS Friendly

Gisquick is set of the mapping applications for publishing QGIS projects. It consists of QGIS plugin, Django based server application, QGIS-server, and web/mobile clients. The process of data publishing with Gisquick is quite easy. You can create and set up your projects with QGIS (layers, metadata, print layouts), prepare it for publishing using Gisquick plugin and upload it to the server. Join QGIS and Gisquick together and gain the ideal solution.

Open for Challenges

Gisquick is developed as an open source project under GNU GPL license. For client-server communication it uses OGC WMS, WMTS and WFS services. Gisquick is the solution without any extra costs (in comparison with common proprietary solutions). It’s applicable in everyday practice (e.g. education or nature conservancy) or for further custom development. Take full advantage of the Open source.

Easy to Deploy

All Gisquick server components can be easily deployed in Docker containers. You can start serving your GIS projects from the cloud (like Amazon Web Services) or from your own hardware without any troubles. For project creators it's even simpler, all they need is QGIS desktop software with Gisquick plugin installed. Deploy Gisquick yourself or use our help.

Accessible Anywhere

Gisquick web client gives you an easy to use user interface for viewing your GIS projects online (via Internet browser on any device). It’s designed as responsive and mobile-friendly application, but a dedicated mobile client (Cordova hybrid app) for even better experience is also available. Use Gisquick to solve your tasks anywhere.
